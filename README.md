# Energymeter-with-GSM

*Company* : Coddec technologies

*Name* : k.likhitha

*Duration* : 4weeks

*Domain* :VLSI and Digital electronics

This project implements a Smart Energy Meter interface using VLSI-based digital logic to measure energy consumption and calculate billing units.
The system simulates GSM transmission of usage data and includes a basic billing module for automated cost computation.
